# Portfolio Website

A personal portfolio website for Yinon Shaul, showcasing skills, projects, and professional experience in Computer Science and Software Development.

## Technologies Used

- **HTML** - Structure and content
- **CSS** - Custom styling and design
- **JavaScript** - Interactive functionality (via Bootstrap)
- **Bootstrap 5** - Responsive UI framework

## Features

- Responsive design that works on all devices
- About section with professional background
- Skills showcase highlighting technical expertise
- Project portfolio with links to GitHub repositories
- Contact information and social media links
- Link to downloadable CV
